# CS300-Portfolio

## Reflection

In this course, I implemented and analyzed multiple data structures to efficiently organize and retrieve course information. Project One focused on evaluating runtime and memory complexity for vectors, hash tables, and binary search trees, while Project Two involved building a program that sorts and displays Computer Science courses in alphanumeric order.

Through these projects, I strengthened my understanding of how data structure selection directly impacts performance, scalability, and search efficiency. I encountered challenges while implementing tree insertion and traversal logic, but resolved them through step by step testing and debugging.

Overall, this work improved my ability to design efficient programs and reinforced the importance of writing clean, maintainable, and adaptable code from the start of development.
